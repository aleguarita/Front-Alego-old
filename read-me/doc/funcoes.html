Funções

Algumas funções são usadas em partes específicas do Front-Alego e outra é para uso dentro seu CSS.

paleta-padrao(cor)
Use para chamar as cores da $paleta-padrao. Exemplo: para definir uma cor como 50% de cinza, use paleta-padrao(cinza-50). Use somente dentro do arquivo config/cores.scss e em mais nenhuma parte do sistema
paleta(cor, tipo, valor)
Esta função é para chamar as cores que você criou na $paleta. Para chamar a cor pura, escreva paleta(nome-cor). Para chamar as variações claras, chame com paleta(nome-cor, clara, $valor). O $valor pode ir de 1 a 100. Para chamar cores escuras chame com paleta(nome-cor, escura, $valor). Use somente dentro do arquivo config/cores.scss
cor-status(cor)
Para chamar as cores de status. Podem ser usadas diretamente no CSS. Veja mais detalhes aqui
cor(tipo, cor)
Principal função para chamar as cores na interface. Cada elemento tem um conjunto de cores a ser chamado, além dos genéricos (conforme já explicado anteriormente). Exemplo: seu texto vai ser um texto destaque. cor(texto, destaque). A cor padrao de cada conjunto não precisa ser chamada, ou seja, o texto padrão seria: cor(texto)
cor-texto(cor-fundo, texto-claro, texto-escuro)
Esta função analisa a cor-fundo declarada e se ela for escura retorna o que você declarou em texto-claro e se for clara, o que você declarou em texto-escuro
breakpoint(medida)
Procura o valor de medida no mapa $breakpoints para retornar um valor para a media query. Veja mais em Sendo responsivo
