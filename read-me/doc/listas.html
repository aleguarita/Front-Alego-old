Listas

Todas as configurações básicas de listas são encontradas no arquivo de configuração. Abaixo você vê todas as configurações.

A lista pode ser chamada através da classe .lista.

.lista
<ul>. Use diretamente no tag de lista para formatá-la
.lista--link
<ul>. Use diretamente no tag de lista de links (como o menu). O link ocupa todo espaço da linha. Caso você precise de algum item da lista (<li>) que não seja link (alguma informação, por exemplo), inclua nele a tag <span>, ficando assim: <li><span>texto</span></li>
.lista__titulo
<li>. Use na primeira linha da lista para ela ganhar o destaque de título
  • $lista--zebra
  • $lista--sombra
