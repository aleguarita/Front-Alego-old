Breadcrumbs

As configurações básicas do breadcrumb são encontradas no arquivo de configuração. Abaixo você vê os tipos disponíveis (única configuração disponível).

Os breadcrumbs são gerados através da gem Breadcrumbrs on Rails.

Quando chamar a gem na view, use a seguinte estrutura:

HTML

<ul class="breadcrumbs">

<%= render_breadcrumbs :tag => :li, :separator => "" %>

</ul>

No HTML gerado no final, cada <li> vai ganhar um link, exceto o da página atual (que é texto puro dentro do <li>, sem nenhum tag adicional).
