Seção Lista

Seção Lista é aquela lista que, ao clicar no título exibe ou oculta o conteúdo (não tive ideia de um nome melhor pra isso). Este arquivo que você está lendo faz uso disso. Cada faixa com texto (como "documentação" ou "Seção Lista escolhida" é um item de seção lista). Muito usada nos sistemas da ALEGO para separar formulário de exibição.

A diferença entre usar a Seção Lista e o mixin lista-oculta é que a Seção Lista já tem um estilo pré-formatado (com direito a seta indicativa que gira) e faz uso do mixin, já o mixin puro é para usar em algum elemento diferenciado.

Para aplicar a seção lista no HTML:

HTML

<section class="secao-lista">

<input type="checkbox" id="id__lista" class="secao-lista__check">

<header class="secao-lista__header">

<label for="id__lista">

<span class="secao-lista__titulo">Título da Lista</span>

<svg class="secao-lista__icone ico"><use xlink:href="ico.svg#seta-fina" /></svg>

</label>

</header>

<section class="secao-lista__conteudo">

Insira aqui o conteúdo

</section>

</section>

Algumas orientações:

  • O id e o for devem ser iguais, evidentemente. E, claro, devem ser únicos.
  • Por padrão a lista aparece aberta. Para que ela apareça fechada por padrão inclua checked="checked" no checkbox.
  • O ícone deve ser inserido através de SVG. Escolha a seta que achar melhor. Veja mais sobre isso
  • $secao-lista-titulo
