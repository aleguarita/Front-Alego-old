Modal

As configurações básicas de cards são encontradas no arquivo de configuração. Abaixo você vê os tipos disponíveis.

A modal pode ser montada com uma série de elementos HTML (como exemplificado abaixo), cada qual com sua classe. Alguns são opcionais (veja a lista de classe para referência).

HTML

<div class="modal">

<div class="modal__janela">

<header class="modal__titulo">

<h1>Título da Modal</h1>

<span class="modal__fechar"><a href=""><svg class="ico"><use xlink:href="ico.svg#fechar" /></svg></a></span>

</header>

<section class="modal__conteudo">

<p>Conteúdo da modal</p>

</section>

<footer class="modal__acoes">

<button class="bt">Aceitar</button>

</footer>

</div>

</div>

As classes abaixo devem ser usadas nos tags HTML como exemplificado acima. Os tags acima são os mais indicados por semântica, mas não obrigatórios (é possível substituir todos por <div>).

.modal
<div>. Formata o fundo da modal e a coloca como flex. A cor de fundo é determinada no arquivo de cores.
.modal__janela
<div>. Formata a janela da modal.
.modal__fechar
<span>. Formata o botão de fechar. Inclua dentro dele um svg com o ícone de fechar (#fechar ou variante)
.modal-conteudo
<section>. Formata o conteúdo da modal.
.modal-acoes
<footer>. Formata o rodapé de ações da modal.
