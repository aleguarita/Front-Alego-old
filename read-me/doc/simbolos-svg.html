Símbolos - SVG

base/simbolos-svg.scss

Os ícones do sistema estão em apenas um arquivo SVG, como se fosse um grande sprite, mas sem a necessidade de usar essa técnica. Para mais detalhes você pode conferir este artigo ou este.

Atenção: só é possível visualizar os SVGs se você estiver rodando em servidor local. Se abrir o HTML direto no browser, o SVG através de símbolo não aparece

Arquivo de ícones

Todos os ícones criados estão individualmente na pasta svg/avulsos para referência e eles aparecem como símbolos dentro do arquivo svg/basico.svg.

Na raíz do Front-Alego tem um arquivo chamado ico.svg que contém todo um conjunto de ícones. Copie e cole os ícones extras do arquivo svg/basico.svg diretamente neste. Exclua ícones não usados para deixar o arquivo final menor. (você sempre poderá copiar algum ícone diretamente do arquivo basico.svg)

Se você for trabalhar variações, coloque no mesmo arquivo. O objetivo no futuro é ter mais de um arquivo de coletânea de símbolos, mas o seu arquivo final sempre deverá ser único.

No Rails coloque o arquivo ico.svg dentro da pasta assets/images.

Chamando um ícone

Para colocar um ícone é necessário utilizar a tag <use> dentro da tag <svg>. Inclua a classe ico (ou variante) dentro da tag <svg>.

Dentro do use você vai usar o parâmetro xlink:href para chamar a imagem (funciona como o href tradicional). No final do nome do arquivo inclua o id da imagem (exemplo: #tabela).

HTML

<svg class="ico">

<use xlink:href="ico.svg#id" />

</svg>
Para colocar um espaço entre o ícone e o texto seguinte, basta colocar um espaço diretamente no HTML.

Classes

As classes abaixo são únicas, não há a necessiade de usar mais de uma na tag <svg> (diferente dos ícones de fonte).

.ico
Tag básica que inclui a formatação básica do ícone (para que ele pegue o tamanho e a cor do texto que o acompanha).
.ico--baixo
Rotaciona o ícone para ele "apontar" para baixo (use com setas).
.ico--cima
Rotaciona o ícone para ele "apontar" para cima (use com setas).
.ico--esquerda
Rotaciona o ícone para ele "apontar" para a esquerda (use com setas).
