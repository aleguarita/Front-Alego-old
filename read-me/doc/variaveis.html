Variáveis

config/_variaveis.scss

Todas as variáveis globais do sistema devem ficar neste arquivo. Ele já vem com um conjunto de variáveis prontas para uso, bastando modificar conforme a necessidade do sistema. Note que diversas delas já estão, inclusive, sendo usadas (por exemplo: $sombra-modal).

$sombra-nenhuma
. Use para eliminar a sombra do elemento (exemplo: o estágio hover de um botão).
$sombra-padrao
. Use como padrão.
$sombra-simples
. Sombra mais discreta para caixas.
$sombra-difusa
$sombra-texto
. Use no texto.
$sombra-texto-relevo
. Use no texto para fazer o efeito de relevo.
$sombra-card
$sombra-modal
$sombra-toaster
$sombra-botao
$borda-nenhuma
. Retira a borda do elemento.
$borda-padrao
. A borda padrão dos elementos.
$borda-variante
. Borda variante (com a outra cor de borda)
$calha
. A variável $grid-margem foi definida em grid e é . Use para dar espaço entre os elementos.
$margem
. Use esta variável como margem padrão dos elementos, gerando consistência. Se necessário use multiplicadores para valores maior (exemplo: margin: $margem * 2;)
$pd
. Use para determinar o padding do objeto. O nome é simplificado para poder usar no padding agrupado (como por exemplo: padding: $pd 0;)
$padding
. Igual ao de cima. Criado apenas para o caso de esquecer a variável acima
$transition
. Animação de transição básica
$rd
. Use como raio de arredondamento padrão (em border-radius). Esta variável tem o nome menos semântico para facilitar na escrita.

Apesar de ter sido usado uma caixa para demonstrar as variáveis, elas podem ser usadas em outros elementos (como texto, por exemplo).
