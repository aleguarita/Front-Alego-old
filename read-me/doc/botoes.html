Botões

Abaixo você verá os tipos de botões e como utilizá-los. Os tipos e algumas configurações especiais podem ser encontradas no arquivo de configuração

São cinco botões à disposição, conforme lista abaixo:

.bt
O botão básico e principal.
.bt--secundario
O botão secundário é usado para ações menos importantes. Se no arquivo de configurações você colocar o mesmo tipo do botão principal, ele terá outra cor
.bt--urgente
Botão de status. Pode ser usado em urgente e erro.
.bt--ok
Botão de status. Use para status de acerto.
.bt--aviso
Botão de status. Use para informações de alerta (amarelo).

No arquivo de configuração você pode desabilitar os botões através do mapa $uso-classes. A opção botao desabilita o .bt e os botões de status. Já a botao-secundario desabilita o .bt--secundario. Se quiser desabilitar da escrita do CSS algum tipo específico de botão de status (ex.: bt--ok) comente a classe no mapa $botoes-status conforme pode ser visto nesta parte do arquivo de configuração

O tipo escolhido para os botões de status é o mesmo do botão secundário

O tamanho dos botões é relativo ao elemento pai (.9em). Teste colocar dentro de um <p> ou direto em um <div> e verá a diferença. Se prefere criar uma lgt;div class="actions"> (como o padrão do Ruby) você pode formatar o tamanho da fonte da própria div, dispensando o uso de p. O tamanho também pode ser alterado acrescentando no botão uma das classes de tamanho de fonte.
