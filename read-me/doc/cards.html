Card

As configurações básicas de cards são encontradas no arquivo de configuração. Abaixo você vê os tipos disponíveis.

Um card pode ser montado com uma série de elementos HTML (como exemplificado abaixo), cada qual com sua classe. Alguns são opcionais (veja a lista de classe para referência).

HTML

<div class="card">

<header class="card__titulo">

<h1>Título do Card</h1>

</header>

<figure class="card__imagem">

<img src="img/card-img.jpg" alt="" />

</figure>

<section class="card__principal">

<div class="card__item">

<p><strong>Teste:</strong> <span>Teste aqui</span></p>

</div>

<div class="card__item">

<p>Teste aqui</p>

</div>

</section>

<footer class="card__footer">

<button class="bt--secundario">Cancela</button>

<button class="bt">Confirma</button>

</footer>

</div>

Para utilizar o item destaque (.card__item--destaque) o HTML seria assim:

HTML

<div class="card__item--destaque">

<p>

<strong>Título</strong> <span>Texto depois do título</span>

</p>

</div>

As classes abaixo devem ser usadas nos tags HTML como exemplificado acima. Alguns tem ordem certa para aparecer, outros não. Os tags listados são os mais indicados por semântica, mas não obrigatórios (é possível substituir todos por <div>).

.card__titulo
<header> Opcional. Formata o título do card. O título pode aparecer depois da imagem.
.card__imagem
<figure> Opcional. Formata a foto ou imagem do card. A imagem pode aparecer antes ou depois do título, a seu critério.
.card__principal
<section> Obrigatório. Formata a parte principal do card, que ajusta a altura quando estiver ao lado de um card maior. Dentro dele você pode colocar o que quiser (p, ul, etc).
.card__item
<div> Opcional. Formata itens separando-os com uma linha (formatação depende do tipo escolhido). Dentro do tag deve ter um <p>. Deve aparecer como filho do .card__principal.
.card__item--destaque
Variante do .card__item acima. Dá um destaque maior para os itens. O título destacado deve aparecer dentro de strong e o texto em span (ver código demonstrativo antes dessa lista).
.card__footer
<footer> Opcional. Formata a área de botões do card.

Se precisar fazer um grupo de cards, use o mixin grupo-flex().
