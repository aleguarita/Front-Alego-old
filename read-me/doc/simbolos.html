Este método vai cair em desuso e está aqui apenas por retrocompatibilidade

Este arquivo contém todos os símbolos usados no Front-Alego e classes para uso com eles.

Lista de Ícones

O mapa $icones lista todos os ícones usados. Para acrescentar um novo olhe o ícone na lista de ícones da fonte e crie o nome da classe. Siga o padrão de organização (sempre em ordem alfabética e cada letra separada por ENTER).

Tamanho do arquivo: este arquivo gera um CSS final imenso pois cada símbolo no mapa cria 5 linhas de CSS. Quando for fazer sites externos, a recomendação é comentar os símbolos não usados (com //) e deixar o arquivo menor. Um comentário ao lado do símbolo mostra quais são as classes que estão sempre em uso, não comente essas.

Classes

.ico-
Todo ícone começa com .ico-. O nome que aparece depois do .ico- é o mesmo que aparece no mapa $icones (por exemplo: .ico-pdf).
.ico--espaco
Acrescenta um espaço depois do ícone (para quando você usa um texto dentro do <span> e manter o ícone separado do texto)

Se você tiver acrescentado algum símbolo novo no seu sistema e quiser visualizá-lo aqui (como referência) vai ser necessário acrescentar o ícone na unha no arquivo read-me/doc.html e copiar uma das linhas existentes para o seu ícone (alterando a classe). Depois disso recompile o read-me/info.scss para read-info.css.

Se na lista abaixo aparecer um "buraco" entre dois ícones é porque o mesmo foi comentado no seu arquivo de símbolos.
