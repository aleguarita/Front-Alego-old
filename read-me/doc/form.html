Formulários

Abaixo você verá os diversos elementos de formulários e como utilizá-los. Os tipos e algumas configurações especiais podem ser encontradas no arquivo de configuração

Para que os formulários sejam formatados direito siga os passos abaixo:

  1. Cada grupo de elemento (label + input) deve estar dentro de uma div. Não pode ter mais de um label e nem mais de um elemento de formulário na div
  2. Essa div vai ter a classe que identifica e formata o elemento (ver mais detalhes abaixo)
  3. Se o código for gerado automaticamente pelo Ruby, retire o <br /> que ele acrescenta depois do label
  4. O uso de <p> dentro do div é desnecessário
  5. O div também pode ganhar a classe de medida field-x conforme exemplicado em Grupo de Formulários
  6. Algumas formatações especiais visuais são feitas colocando o label depois do input. Apesar de aumentar a experiência de uso, pode prejudicar quem usa leitores de tela. Use este recurso com cuidado.

Configurações

No arquivo de configuração você determina algumas configurações além do tipo.

input type="text" (e equivalentes) e textarea

Classe do elemento pai
.form-texto
Elementos que formata
Caixas de texto do CSS 3 (principalmente <input type="text">) e textarea

A classe .form-texto é bem genérica e pode ser usada para qualquer caso de formulário que não sejam casos mais específicos (como radio, select).

No HTML o label pode aparecer depois do input/textarea. Quando isso acontece, visualmente ele vai aparecer antes e vai ganhar uma formatação visual de destaque quando o input estiver ativo. Evite usar isso em casos de que o usuário pode usar um leitor de tela.

É possível acrescentar um ícone depois da caixa (por exemplo, para campos de busca). Para isso você pode acrescentar um span ou button e acrescentar a classe .form__ico, dentro deste elemento coloque o símbolo desejado através de SVG.

Código básico de um elemento de texto com o label depois (no exemplo clique na caixa de texto para ver o efeito).

HTML

<div class="form-texto">

<input type="text" id="exemplo-texto-1" />

<label for="exemplo-texto-1">Exemplo</label>

</div>

Um exemplo de como usar o ícone para fazer um campo de busca (neste exemplo não uso label e sim placeholder):

HTML

<div class="form-texto">

<input type="text" placeholder="Busca" id="exemplo-texto-2" />

<span class="form__ico"><svg class="ico"><use xlink:href="ico.svg#lupa" /></svg></span>

</div>

select

Classe do elemento pai
.form-select
Elementos que formata
select

Código exemplo para um select:

HTML

<div class="form-select">

<label for="exemplo-select">Select padrão</label>

<select id="exemplo-select">

<option value="val1">Exemplo 1</option>

<option value="val2">Exemplo 2</option>

<option value="val3">Exemplo 3</option>

</select>

</div>

radio button e checkbox

Classe do elemento pai
.form-radio
Elementos que formata
radio button
Classe do elemento pai
.form-checkbox
Elementos que formata
checkbox

Cada elemento tem sua configuração. Cada div deve contér apenas um checkbox ou radio button. Fora isso não é necessário mais nenhuma classe ou html.

É possível decidir se esses elementos aparecerão como o formato padrão do browser ou personalizado pelo Front-Alego (valor padrão, claro).

Checkbox e radio não funcionam muito bem ao lado de outros elementos se a opção $label-tipo for flutuante

Código para o radio button:

HTML

<div class="form-radio">

<input type="radio" id="campo1" name="exemplo" checked="checked" />

<label for="campo1">Exemplo radio button</label>

</div>

<div class="form-radio">

<input type="radio" id="campo2" name="exemplo" />

<label for="campo2">Exemplo radio button</label>

</div>

<div class="form-radio">

<input type="radio" id="campo3" name="exemplo" />

<label for="campo3">Exemplo radio button</label>

</div>

Código para o checkbox:

HTML

<div class="form-checkbox">

<input type="checkbox" id="campo1b" name="exemplo" checked="checked" />

<label for="campo1b">Exemplo checkbox</label>

</div>

<div class="form-checkbox">

<input type="checkbox" id="campo2b" name="exemplo" />

<label for="campo2b">Exemplo checkbox</label>

</div>

<div class="form-checkbox">

<input type="checkbox" id="campo3b" name="exemplo" />

<label for="campo3b">Exemplo checkbox</label>

</div>
