Tabelas

Todas as configurações básicas de tabela são encontradas no arquivo de configuração. Abaixo você vê todas as configurações.

Para chamar uma tabela você tem duas classes disponíveis (desde que ativas em $uso-classes, conforme descrito no arquivo de configuração):

.tab
<table>. A classe básica para todas as tabelas
.tab--form
<table>. Use para tabelas formulários (estilo planilha)

Caso precise de variantes ou de modificar a classe básica para atender à necessidade do layout, faça-o em um arquivo tabela.scss dentro da pasta extra/.

Existem algumas classes para serem usadas dentro das tabelas:

.tab__acoes
<td>. Use no <td> que contém os ícones de ações
.col--X
<th>. Use nas colunas de título (<th>). X é substituído pelo nome que você criou dentro do arquivo de config e determina a largura desta coluna. No momento já temos criada a .col--acoes
  • $tab-zebra
  • $tab-acao-link-bloco
  • .tab--form
