Abas

As configurações básicas da abas são encontradas no arquivo de configuração. Abaixo você vê os tipos disponíveis.

Para criar abas que exibem e ocultem o conteúdo é necessário deixar o HTML bem estruturado ou não vai dar certo.

Você também vai notar que existem várias classes que devem receber um número. Lembre-se que o maior número disponível da classe é configurado no arquivo de configuração através da classe $abas-qtd.

Siga os passos:

  1. Crie uma section e acrescente a classe abas.
  2. Essa section é dividida em três partes: radio buttons, nav e conteúdo. Estes três grupos devem estar no mesmo nível.

    3. Até o momento será algo como:

    HTML

    <section class="abas">

    <!-- inputs radio button -->

    <!-- nav - área onde o usuário escolhe que conteúdo ver -->

    <!-- conteúdo das abas -->

    </section>
  3. radio buttons: eles devem ser filhos diretos do <section class="abas">. Você deverá ter um radio para cada aba.
  4. Coloque um name para todos os radios. Como padrão sugiro abas.
  5. Cada radio deve ter um id que seria aba-x e as classes abas__input e aba-x (substitua x pelo número da aba).
  6. Defina qual a aba padrão para ser a ativa e acrescente checked="checked" no radio específico.

    7. Se tivermos três abas como exemplo com a primeira aba sendo exibida por padrão, o código seria:

    HTML

    <section class="abas">

    <input type="radio" id="aba-1" class="abas__input aba-1" name="abas" checked="checked">

    <input type="radio" id="aba-2" class="abas__input aba-2" name="abas">

    <input type="radio" id="aba-3" class="abas__input aba-3" name="abas">

    <!-- nav - área onde o usuário escolhe que conteúdo ver -->

    <!-- conteúdo das abas -->

    </section>
  7. navegação: abra um tag nav com a classe abas__nav.
  8. Dentro dela coloque os labels para os radio.
  9. Cada label deve ter o for equivalente ao id do radio que você criou anteriormente e deve ter as classes abas__aba e aba-x (substitua x pelo número do label, que deve ser igual ao id e classe do radio).

    10. No momento, com três abas teríamos:

    HTML

    <section class="abas">

    <input type="radio" id="aba-1" class="abas__input aba-1" name="abas" checked="checked">

    <input type="radio" id="aba-2" class="abas__input aba-2" name="abas">

    <input type="radio" id="aba-3" class="abas__input aba-3" name="abas">

    <nav class="abas__nav">

    <label for="aba-1" class="abas__aba aba-1">Título da aba 1</label>

    <label for="aba-2" class="abas__aba aba-2">Título da aba 2</label>

    <label for="aba-3" class="abas__aba aba-3">Título da aba 3</label>

    </nav>

    <!-- conteúdo das abas -->

    </section>
  10. abas: cada aba deve estar no mesmo nível. Use a tag article para cada uma delas.
  11. Cada aba deve ter uma classe abas__conteudo e a classe aba-x que deve ser igual à do label e radio.

    12. O HTML final ficaria:

    HTML

    <section class="abas">

    <input type="radio" id="aba-1" class="abas__input aba-1" name="abas" checked="checked">

    <input type="radio" id="aba-2" class="abas__input aba-2" name="abas">

    <input type="radio" id="aba-3" class="abas__input aba-3" name="abas">

    <nav class="abas__nav">

    <label for="aba-1" class="abas__aba aba-1">Título da aba 1</label>

    <label for="aba-2" class="abas__aba aba-2">Título da aba 2</label>

    <label for="aba-3" class="abas__aba aba-3">Título da aba 3</label>

    </nav>

    <article class="abas__conteudo aba-1">

    Conteúdo da aba 1

    </article>

    <article class="abas__conteudo aba-2">

    Conteúdo da aba 2

    </article>

    <article class="abas__conteudo aba-3">

    Conteúdo da aba 3

    </article>

    </section>

Lista de classes usadas:

.abas
<section>. A classe principal para o elemento pai
.abas__x
<input type="radio">, <label>, <article>. Essa classe é que formata as condições especiais de quando seleciona uma aba (destacando item de navegação e exibindo conteúdo). x deve ser substituído pelo número da aba e cada elemento deve ser correspondente.
.abas__nav
<nav>. A navegação das abas.
.abas__input
<input type="radio">. Serve para ocultar o input.
.abas__aba
<label>. Formata os itens de navegação.
.abas__conteudo
<article>. Formatação do conteúdo (que estará oculto por padrão).
