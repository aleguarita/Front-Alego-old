Tipografia

Fontes: base/_fontes.scss

Configuração: config/_tipografia.scss

A tipografia é ajustada em dois arquivos separados. Um na pasta base/ (que não deve ser alterado) e outro na pasta config (feito para alteração).

Não coloque muita formatação dentro de tags básicos como <h1> ou <a>. Dê preferência para uso de classes ou seletores descendentes para formatações complexas (exemplo: .artigo-padrao h1)

Chamando fontes

Dentro do arquivo base/_fontes.scss tem as variáveis para as principais fontes a serem usadas.

$fonte-serifa
Fonte com serifa: variação da Patua One (se não usar tipografia personalizada, será a Palatino)
$fonte-sem-serifa
Fonte sem serifa: OpenSans (se não usar tipografia personalizada, será a Helvetica)
$fonte-monoespacada
Fonte monoespaçada: Consolas
$fonte-simbolo
Fonte de símbolos da Assembleia

Configurações: variáveis

As três primeiras variáveis do arquivo config/_tipografia.scss determinam configurações básicas do texto:

$fonte-titulo
Qual a fonte padrão para os títulos do sistema. Por padrão é a $fonte-serifa
$margem-titulo
Define a margem dos títulos <h1> até <h6>
$margem-texto
Define a margem dos elementos de texto e similares. Incluindo tabelas e listas

As três variáveis seguintes determinam o tamanho do texto baseado no modular-scale do Bourbon.

$modular-scale-ratio
Modifica o valor padrão estabelecido pelo Bourbon.
$modular-font-size
O tamanho base a ser usado pelo modular-scale(). Se você quiser que o sistema todo tenha uma fonte maior ou menor, altere aqui
$tamanho-fonte-padrao
A variação do tamanho de texto padrão baseado no modular-scale()
Estas variáveis estão no arquivo de configuração ( config/_tipografia.scss) para que você possa alterar conforme a necessidade do sistema atual.

Formatação personalizada

Sempre que for criar estilos mais personalizados (como com uso de bordas e sombras), procure utilizar classes ou seletores descendentes (lembre-se de não abusar de elementos descendentes. Mantenha a regra de no máximo um nível, se for necessário mais do que isso, é adequado o uso de classe).

Existe disponível alguns mixins para variações de título, texto e link.

Padrões de texto

O mixin texto() altera as propriedades básicas dos textos. Este mixin é puxado no arquivo de reset e qualquer alteração destas propriedades vai alterar todo o seu site/sistema, portanto, evite alterar. É possível acrescentar mais propriedades, para isso acrescente o seguinte fragmento de texto antes da última chave (}):

SASS

@else if $mod == nome-variante {

Regras

}

Para usar este mixin (se precisar) chame com @include texto($mod) e coloque o nome do tipo no lugar de $mod:

padrao
Configurações básicas do corpo de texto, listas e tabelas.
titulo
Configuração básica do texto dos títulos.

Demais mixins

Estes são mixins para usar em classes ou seletores descedentes que formatam texto. Tem um para título (com apenas um variante) e três para links. Este mixin não está sendo usado ativamente e está disponível para seu uso ou alteração. Para usar o mixin ou acrescentar variantes neles siga as instruções acima.

titulo()

Use este mixin para configurações especiais de título, que fujam do padrão. Até o momento o Front-Alego oferece apenas uma opção: borda.

borda
Aplica uma borda na parte inferior do título, que fica com a mesma cor do texto.

link()

Use este mixin para configurações especiais de link e seus variantes. Algumas delas já usadas.

sem-sublinhado
Tira o sublinhado do link e aplica quando passa ou mouse por cima
borda
Tira o sublinhado e substitui por uma borda fina e cinza. Quando passa o mouse por cima, o sublinhado fica da cor do link (que não muda de cor).
pontilhado
Igual ao de cima, mas com a diferença que a borda é pontilhada e o link escurece ao passar por cima
