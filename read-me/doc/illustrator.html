Illustrator

O layout deve ser gerado no Illustrator prioritariamente. Existe um arquivo modelo pronto para isso. Neste arquivo existem layers prontas com os elementos além das paletas igual à usada no sistema atual.

As cores da paleta são globais e alteram os elementos automaticamente. Para criar uma nova paleta siga o mesmo padrão, inclusive de nome, para que possa ser transcrito no arquivo de configuração do Front-Alego.

Quando alterar ou criar uma nova paleta, comece pela quarta cor (a cor do meio) de cada paleta e crie as três variantes mais claras (as 3 primeiras) e as três variantes mais escuras (as 3 últimas).
