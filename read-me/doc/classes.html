Classes globais

base/_classes.scss

Além das inúmeras classes geradas em vários outros arquivos, o arquivo base/_classes.scss contém várias classes globais comum a todos os sistemas da Alego. Em seu projeto, quando criar classes genéricas (a exemplo da .oculta), faça dentro de um arquivo _classes.scss na pasta extra.

.oculta
Some com o elemento que usar esta classe. Usada, normalmente com script que acrescenta essa classe
.pointer
O elemento ganha o cursor de link quando passa o mouse por cima. Passe o mouse por cima deste texto para ver
