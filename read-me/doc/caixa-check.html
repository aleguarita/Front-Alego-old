Caixas clicáveis

A caixa clicável é uma que tem dois estados: não selecionada e selecionada. Um exemplo seria a caixa que você vê ao lado.

Para criar este efeito é preciso utilizar o mixin item-checado (Veja mais detalhes em mixins).

O HTML

Você pode aplicar isso em itens de uma lista (por exemplo, quando há multiplas opções) ou em divs separados. Já o conteúdo deste elemento deve ter:

  1. Um checkbox (óbvio)
  2. Um label no mesmo nível hierárquico que o checkbox e depois dele
  3. O conteúdo deve aparecer dentro do label

O código HTML da caixa ao lado é:

HTML

<div class="caixa-clicavel">

<input class="caixa-clicavel__check" type="checkbox" id="cc-teste">

<label class="caixa-clicavel__label" for="cc-teste">

<p><svg class="ico"><use xlink:href="../svg/basico.svg#monitor" /></svg></p>

<p>Clique em qualquer lugar dessa caixa e veja o que acontece</p>

</label>

</div>

O SASS

  1. O elemento pai deve ter o mínimo de formatação possível (como tamanho, por exemplo)
  2. Formate o estado padrão do label, quando o checkbox não está selecionado (faça isso antes do checkbox, por boas práticas)
  3. O label deve ter display: block para que ele ocupe toda a área do elemento pai
  4. No checkbox chame o mixin item-checado(). Entre parênteses você vai colocar o seletor do label (pode ser o nome da classe ou só label mesmo. Importante notar que deve estar entre aspas). Formate este mixin como se fosse um seletor, abrindo chaves e colocando aí o conteúdo que vai aparecer quando o checkbox estiver selecionado

O código SASS da caixa ao lado é:

SASS

.caixa-clicavel {

max-width: 200px;

float: right;

&__label {

display: block;

border: 1px solid cor(borda);

p {

padding: $calha / 2;

text-align: center;

margin: 0;

&:nth-child(1) {

background: cor(fundo);

font-size: 3em;

color: cor(texto, claro);

}

}

}

&__check {

@include item-checado(".caixa-clicavel__label") {

outline: 1px solid cor(borda);

box-shadow: 0 0 30px cor(fundo);

p:nth-child(1) {

background: cor(fundo, escuro);

}

}

}

}

Compilado este código vira:

CSS

.caixa-clicavel {

max-width: 200px;

float: right;

}

.caixa-clicavel__check {

display: none;

}

.caixa-clicavel__check:checked ~ .caixa-clicavel__label {

outline: 1px solid #00c5cd;

box-shadow: 0 0 30px #00c5cd;

}

.caixa-clicavel__check:checked ~ .caixa-clicavel__label p:nth-child(1) {

background: #00949a;

}

.caixa-clicavel__label {

display: block;

border: 1px solid #00c5cd;

}

.caixa-clicavel__label p {

padding: 0.3125rem;

text-align: center;

margin: 0;

}

.caixa-clicavel__label p:nth-child(1) {

background: #00c5cd;

font-size: 3em;

color: #fff;

}
